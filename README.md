A premium, modern web portfolio crafted for an interior designer. The design focuses on elegance, warmth, and smooth interactivity to showcase spaces and creative arts beautifully. 

> Originally inspired by [Figma Design](https://www.figma.com/design/OYgBVBKO30xlrVcNlbzV3p/Interior-Designer-Portfolio).

## ✨ The Aesthetic

The aesthetic of this portfolio is meticulously designed to reflect the taste and sophistication of high-end interior design:

- **Earthy & Warm Color Palette:** Built around a harmonious blend of creamy beiges (`#E0D6B8`), deep espresso browns (`#2C1E1B`), warm terracottas (`#8B453E`), and muted golds (`#B8842C`). This palette evokes a sense of natural materials like wood, leather, and stone.
- **Elegant Typography:** A sophisticated typographic pairing utilizing **Fraunces** (a refined, old-style serif for dramatic, large display headings) and **Jost** (a clean, geometric sans-serif for easily readable, tracked-out uppercase accents and body copy).
- **Fluid Micro-Interactions:** Leverages Framer Motion to create smooth, curtain-like page transitions, parallax scrolling effects on imagery, and delicate word-by-word text reveals that make the interface feel alive and polished.
- **Premium Layouts:** Extensive use of whitespace, dramatic scale contrasts, and elegant borders give the content room to breathe, framing imagery much like an art gallery.

## 🚀 Features

- **Immersive Hero Section:** Parallax background image with a subtle gradient overlay and smooth entrance animations.
- **Dynamic Page Transitions:** Seamless navigation between pages using `AnimatePresence`.
- **Responsive Design:** Fully responsive layout that looks stunning on mobile, tablet, and desktop devices.
- **Custom Loading Screen:** Enhances perceived performance and maintains the premium feel from the first click.
- **Modern Tech Stack:** Built for speed and maintainability using Vite, React, and Tailwind CSS v4.

## 🛠️ Tech Stack

- **Framework:** [React 18](https://react.dev/) + [Vite](https://vitejs.dev/)
- **Styling:** [Tailwind CSS v4](https://tailwindcss.com/)
- **Animations:** [Framer Motion (motion/react)](https://motion.dev/)
- **Routing:** [React Router v7](https://reactrouter.com/)
- **UI Components:** Radix UI Primitives & custom handcrafted components

## 📦 Getting Started

Follow these steps to run the project locally.

### Prerequisites
- Node.js (v18 or higher recommended)
- npm or pnpm

### Installation

1. Clone the repository or navigate to the project directory:
   ```bash
   cd "Interior Designer Portfolio"
   ```

2. Install the dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

4. Open your browser and visit the local URL provided by Vite (usually `http://localhost:5173`).

## 🏗️ Building for Production

To create a production-ready build:

```bash
npm run build
```
The optimized files will be generated in the `dist` directory.
